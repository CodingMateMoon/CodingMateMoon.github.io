오라클의 데이터 구조를 창고에 비유했을 때 테이블스페이스라는 창고 안에는 세그먼트라는 다층 수납 선반이 있고 이 선반 하나 하나를 익스텐트라고 하며 이 선반안에는 여러 데이터 물건들을 포장한 박스 블록들이 존재하는 것과 비슷하다고 볼 수 있습니다. 

블록 집합 -> 익스텐트
익스텐트 집합-> 세그먼트
세그먼트 집합-> 테이블스페이스
테이블스페이스는 1개 이상의 데이터 파일로 구성.
